# Assembly: Endgame 🎮

A fun word-guessing game built with **React** and **Vite** where the objective is to guess the hidden word before all programming languages are eliminated, leaving only Assembly.

## 🚀 Features

- Random word generated every game
- Interactive on-screen keyboard
- Correct and incorrect letter highlighting
- Tracks remaining programming languages
- Win/Loss detection
- New Game functionality
- Responsive UI
- Confetti animation on victory

## 🛠️ Built With

- React
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- React Confetti

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/assembly-endgame.git
```

Move into the project directory

```bash
cd assembly-endgame
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```

## 🎯 How to Play

- Guess the hidden word one letter at a time.
- Each incorrect guess eliminates one programming language.
- You have **8 incorrect attempts**.
- Guess all unique letters before Assembly is the only language left!
